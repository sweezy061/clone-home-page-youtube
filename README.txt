 Clone do YouTube

 Objetivo do Trabalho

 O objetivo deste projeto foi recriar a interface principal do site YouTube de forma estática,
 utilizando apenas HTML e CSS.
 O foco é a prática de estruturação de páginas web, organização de layout, estilização e
 responsividade básica, sem funcionalidades dinâmicas.
 Site Real de Referência

 YouTube Oficial: https://www.youtube.com/

 Tecnologias Utilizadas

 HTML5 → estrutura da página
 CSS3 → estilização e layout responsivo
 Imagens → capturas e recursos visuais semelhantes ao site original
